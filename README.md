# SEM6 Project Onepager

## 📌 Projectbeschrijving

De **SEM6 Project Onepager** is een webpagina die is ontwikkeld om informatie over het semester 6 project te presenteren en te delen met een externe projectpartner.

De onepager fungeert als een centrale plek waar **updates, voortgang en belangrijke informatie** over het project overzichtelijk worden weergegeven. Hierdoor kan de projectpartner eenvoudig de nieuwste ontwikkelingen volgen zonder meerdere documenten of platforms te hoeven raadplegen.

Alle informatie staat op één webpagina, waardoor de inhoud snel en overzichtelijk te bekijken is.

---

## 🎯 Doel van de applicatie

Het doel van deze onepager is om:

- projectinformatie duidelijk te presenteren
- de projectpartner op de hoogte te houden van de voortgang
- nieuwe updates en ontwikkelingen eenvoudig te delen
- alle relevante informatie op één centrale plek beschikbaar te maken

De website wordt gedurende het semester bijgewerkt met **nieuwe informatie, voortgang en projectresultaten**.

---

## 🚀 Functionaliteiten

De applicatie bevat onder andere:

- Een overzichtelijke **one-page layout**
- Secties met **projectinformatie en updates**
- Een duidelijke structuur zodat informatie gemakkelijk te vinden is
- Responsief design (werkt op desktop, tablet en mobiel)

---

## 🛠 Technologieën

Dit project is ontwikkeld met behulp van moderne webtechnologieën:

- **HTML5** – Voor de structuur van de webpagina
- **CSS3** – Voor styling en layout
- **JavaScript** – Voor interactieve functionaliteiten
- **Node.js** – Voor het draaien van de development omgeving
- **npm** – Voor het beheren van dependencies

Afhankelijk van het project kunnen frameworks of libraries worden gebruikt zoals:

- **React**
- **Vite**
- **Tailwind CSS**

## 📦 Installatie

Volg onderstaande stappen om het project lokaal te draaien.

### 1. Installeer Node.js

Download Node.js via:
https://nodejs.org

Dit installeert ook **npm (Node Package Manager)**.

---

### 2. Clone de repository

Clone de repository naar je computer:

```
git clone https://github.com/Chrisssss15/sem6-onepager.git
```

Of download het project als **ZIP** via GitHub.

---

### 3. Ga naar de projectmap

Open een terminal en ga naar de projectmap:

```
cd sem6-onepager
```

---

### 4. Installeer dependencies

Installeer alle benodigde packages:

```
npm install
```

---

### 5. Start de applicatie

Start de development server:

```
npm run dev
```

De applicatie wordt lokaal gestart.

Open vervolgens in je browser:

```
http://localhost:5173
```
